# 🌙 Lua: Do Zero ao Avançado

Um livro completo de Lua em português, com 150+ capítulos práticos.

🌐 **Online:** https://wallysondevs.github.io/lua-book/

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- Wouter (hash routing — funciona no GitHub Pages)
- Framer Motion + Lucide Icons

## Desenvolvimento

```bash
npm install
npm run dev
npm run build
```

## Deploy (GitHub Pages)

```bash
npm run build
npm run deploy   # publica dist/ na branch gh-pages
```
